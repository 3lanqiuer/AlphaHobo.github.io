找出最大的id
发表于2024-06-05|实用小函数合集
1let maxId = data.reduce((max, row) => (row.id > max ? row.id : max), 0); reduce 方法接受两个参数：一个回调函数和一个初始值。 回调函数 (max, row) => (row.id > max ? row.id : max)： max 是当前的累积值，初始为 0（由第二个参数提供）。 row 是当前遍历到的数组元素。 如果 row.id 大于 max，则返回 row.id，否则返回 max。 示例： 初始值 max 为 0。 第一次迭代：row.id 为 3，比较 3 和 0，更新 max 为 3。 第二次迭代：row.id 为 7，比较 7 和 3，更新 max 为 7。 第三次迭代：row.id 为 5，比较 5 和 7，max 保持为 7。 最终，maxId 的值为 7。
筛选唯一数据对象
发表于2024-05-30|实用小函数合集
1234const uniqueData = data.filter( (item, index, self) => self.findIndex((t) => t.extend === item.extend) === index); item：当前处理的元素，即 data 数组中的每个对象。 index：当前元素在 data 数组中的索引。 self：调用 filter 方法的原数组，在这里就是 data。 查找当前元素 item 的 extend 值在数组中第一次出现的位置，并与当前索引进行比较。如果这个位置等于当前索引，则说明这是该 extend 值第一次出现，可以保留；否则，就说明这个 extend 值已经出现过，不保留。 这就实现了仅保留 extend 字段唯一的对象的效果。
